for es una funcion de javascript para poder repetir algo mientras que yo condicione que se repita.

&& es un operador logico para decir 'Y' por ejmplo 'Esto y esto'

|| es un operador logico para decir 'Esto o Esto otro'

! es un operador logico para decir 'Esto no, sino lo opuesto' ejemplo: !VERDADERO = FALSO
